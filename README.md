## :closed_book: 배포

* AWS EC2 - 클라우드 컴퓨팅 시스템
* AWS elastic beanstlak - 서버 배포및 관리 프로비저닝 서비스
* AWS S3 - 클라우드 데이터 저장소
* Atlas - MongoDB 클라우드 호스팅 서비스
* Docker - 컨테이너 기반 가상화 소프트웨어 플랫폼
* Nginx - 프록시 서버 (보안 향상 및 캐시를 활용한 전송 속도 향상)
